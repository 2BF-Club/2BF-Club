# 2BF-Club
2BF层友会主页

## Tips
- 如果要添加新的`moments`，需要在`moments.html`中的`events`变量中添加对应的信息，如果有照片，在`/public/img/`中添加图片
- 如果要添加新的`paper`，只需要在`publications.html`中的`publications`变量中添加对应的信息即可
